## 리눅스 커맨드라인 기초

1. pwd : print working dorectory
    - 탐색기에서 내가 현재 작업하고 있는 경로 파악 가능

2. cd : change directory 
    - 더블클릭해서 폴더 접속하는 것과 같은 역할
    

3. ls : list
    - 현재 작업 경로의 파일 목록 파악 가능
        - ls -a : 숨긴 파일도 파악 가능
        - ls -l : 파일의 자세한 내용도 파악 가능
        - ls -al : 두개 다 볼 수 있음
